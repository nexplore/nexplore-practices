Nexplore Practices Tailwind UI Component Library

Welcome to the Nexplore Practices Tailwind UI Component Library sample application. This library provides a comprehensive set of Angular components, styled using a customizable Tailwind theme.

The design system has been adapted based on the Kanton Bern Design System

Browse through the examples below to see the components in action.

 }

Installation & Requirements

Install with npm:

npm install @nexplore/practices-ui-tailwind

Requirements:

  • Angular 19+
  • Tailwind CSS 4.x
